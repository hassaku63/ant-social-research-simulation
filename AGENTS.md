# AGENTS.md — エージェント向け指示（OpenAI Codex / 汎用AIエージェント）

このファイルはOpenAI CodexおよびAI汎用エージェントが常時参照するシステムレベルの指示です。

> **メンテナンス注意**: このファイルの内容は [`docs/ai-guidelines-master.md`](docs/ai-guidelines-master.md) をマスターとしています。
> 内容を更新する場合は必ず全AIサービス向けファイルを同時に更新してください。

## リポジトリの目的

社会性昆虫（蟻・蜂）の集団知性（Swarm Intelligence）を研究・応用するリポジトリ。

- **研究テーマ**: 蟻・蜂の行動システムに関する先行研究を収集・整理し、人間社会やソフトウェアへの応用事例を調査・ドキュメント化する
- **実装テーマ**: 蟻・蜂の行動システムを応用できる実問題を検討し、設計・実装・検証を反復する

## 作業の原則

### タスクサイクル（必ず遵守）

すべての作業は以下のサイクルを反復する:

1. **計画の言語化** — 何を達成するか、スコープと成果物を明文化する
2. **詳細化（仕様検討）** — 具体的な手順・仕様・評価基準を定める
3. **実行（実装）** — 計画に従い実装・文書作成を行う
4. **検証** — 成果物が評価基準を満たすかテスト・確認する
5. **振り返りを記録** — 学んだこと・課題・次のアクションを記録する

### コード変更のルール

- 変更は最小限かつ目的に特化したものにする
- 新規実装には対応する仕様ドキュメント (`docs/simulation/`) が存在することを確認する
- テストが存在する場合は変更後に必ず実行する
- コミットメッセージはタスクの目的を明確に記述する

### ドキュメント変更のルール

- 既存ドキュメントと重複する内容は差分・拡張として明示する
- 調査結果は出典（論文・URL等）を必ず記載する
- 振り返り内容は各タスクドキュメントに追記し蓄積する
