# プロジェクト概要 (Amazon Kiro Steering)

このファイルはAmazon Kiroが常時参照するステアリングファイルです。
`inclusionMode: always` として扱われ、全セッションを通じて有効です。

## リポジトリの目的

社会性昆虫（蟻・蜂）の集団知性（Swarm Intelligence）を研究・応用するリポジトリ。

蟻・蜂は、個体としては単純なルールに従いながら、集団として驚くほど効率的な行動を実現する。
このメカニズムを深く理解し、人間社会やソフトウェアへの応用を探ることが目的。

## 2つの研究テーマ

### テーマ1: 先行研究調査

- 蟻・蜂の行動システムに関する先行研究を収集・整理する
- 人間社会やソフトウェアへの応用事例を調査する
- なぜ・いかに機能するかを分析し、ドキュメントとしてまとめる
- **成果物保存先**: `docs/research/`

### テーマ2: シミュレーション・応用実装

- 蟻・蜂の行動システムを応用できる実問題を検討する
- コンセプト検討→計画→仕様策定→実装→検証のサイクルを反復する
- **仕様保存先**: `docs/simulation/` / **コード保存先**: `src/`
- **Kiro仕様ファイル保存先**: `.kiro/specs/`

## タスクサイクル（全作業で遵守）

```
計画の言語化 → 詳細化（仕様検討）→ 実行（実装）→ 検証 → 振り返りを記録
```

Kiroでタスクを実行する際は、新しいスペックファイルを `.kiro/specs/<task-name>.md` に作成し、
上記サイクルの各ステップを記録すること。

## ディレクトリ規約

| パス | 用途 |
|------|------|
| `docs/research/` | テーマ1の調査ドキュメント |
| `docs/simulation/` | テーマ2の設計・仕様ドキュメント |
| `docs/workflow/` | タスク進行ガイドライン・テンプレート |

| `.kiro/specs/` | Kiro向け機能仕様・タスク管理 |
