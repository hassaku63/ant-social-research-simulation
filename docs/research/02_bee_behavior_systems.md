# 蜂の行動システム

## 概要

蜜蜂（Apis mellifera など）は、蟻と並んで最も研究されている社会性昆虫の一つである。蜂のコロニーは数万匹の個体で構成され、高度に協調した採餌行動、巣の温度管理、新女王の選出など、多様な集団的意思決定を行う。

---

## 主要な行動メカニズム

### 1. ワグルダンス（Waggle Dance）

**発見者:** Karl von Frisch（1973年ノーベル賞受賞）

**概要:**  
ミツバチが食料源（花蜜、花粉）や新しい巣の場所を仲間に伝えるために行う「踊り」。

**ダンスの構造:**

```
    ↑ (上方向 = 太陽方向)
    |
  ← | →
    |
    ↓
```

- **直進部分（waggle run）の向き:** 太陽の方向を基準に食料源の方向を示す
  - 上方向 = 太陽の方向と同じ
  - 角度のズレ = 太陽方向から見た食料源の角度
- **直進部分の持続時間:** 食料源までの距離に比例
  - 1秒 ≈ 約1km
- **ダンスの活発さ:** 食料源の品質（糖濃度など）に比例

**情報の圧縮と伝達:**  
3次元の情報（方向、距離、品質）を2次元の踊りとして表現する高度な情報符号化システム。

---

### 2. 採餌行動のロールスイッチ

**スカウト蜂とフォロワー蜂:**

| 役割 | 行動 |
|------|------|
| スカウト蜂（Scout bee）| 自発的に新しい食料源を探索する |
| フォロワー蜂（Follower bee）| スカウト蜂のダンスを見て食料源に向かう |

**動的なバランス:**
- 良い食料源が見つかると、スカウト蜂はより活発にダンスをし、多くのフォロワーを誘引
- 食料源が枯渇すると、ダンスが弱まり蜂が自然と離れる
- コロニー全体として、複数の食料源の中から最良のものに資源を集中させる

---

### 3. 蜂の巣の意思決定（新女王蜂の選出・分蜂先の選択）

**分蜂（Swarm）時の集団意思決定:**

分蜂とは、女王蜂が古いコロニーの半数の働き蜂を連れて新しい巣を作るプロセス。

**新しい巣場所の選定プロセス（Thomas Seeley の研究）:**
1. スカウト蜂が分散して候補の巣場所を探索
2. 各スカウト蜂は候補地の品質（広さ、入口の大きさ、隙間の少なさなど）を評価
3. 良い候補地を見つけたスカウト蜂はダンスでその場所を宣伝（品質が高いほど激しく踊る）
4. 各スカウト蜂は他のスカウト蜂のダンスを見て候補地を調べ、より良い場所のダンスに乗り換える
5. 一つの候補地に支持が集まると「合意」に達し、コロニーが新しい巣に移動する

**この意思決定の特徴:**
- 全員一致を必要としない（しきい値に達した時点で移動）
- ネガティブなフィードバックを使って合意を加速（停止信号：良くない選択肢のダンスを止めさせる行動）
- 品質の高い選択肢に収束する（質の悪い候補地を支持するスカウトは自然に減少する）

---

### 4. 温度管理と集団体温調節

**巣内温度調節:**  
蜂は巣内の温度を約34〜35℃に保つ（幼虫の発育に最適）。

| 状況 | 行動 |
|------|------|
| 過熱時 | 翅を扇いで換気、水を巣内に持ち込んで蒸発冷却 |
| 低温時 | 集合して体温を共有（クラスター形成）、飛翔筋を振動させて発熱 |

**分散型温度制御:**  
どの蜂も巣全体の温度を直接測定していないが、局所的な温度（自分の周囲）への反応が全体の温度調節に寄与する。

---

### 5. 蜂の分業とカースト

| カースト | 役割 |
|---------|------|
| 女王蜂 | 産卵・コロニーの維持（コロニーに1頭） |
| 雄蜂（Drone） | 交尾（受精後は不要となり排除されることもある） |
| 働き蜂（Worker） | 採餌、育児、建設、防衛、女王の世話 |

**働き蜂の日齢多型（Age Polyethism）:**  
働き蜂は成長に伴い担当する仕事が変化する。

| 日齢 | 担当する仕事 |
|------|-------------|
| 1〜3日 | 巣内の清掃 |
| 4〜11日 | 幼虫の世話（乳母蜂） |
| 12〜17日 | 蜜の受け取り・貯蔵、ロウの分泌で巣の建設 |
| 18〜21日 | 入口の警備 |
| 22日以降 | 採餌活動 |

## 主要な研究成果

### Karl von Frisch（1967）- ワグルダンスの解読
- 著書 "The Dance Language and Orientation of Bees" でワグルダンスが方向と距離の情報を符号化することを実証
- 1973年ノーベル生理学・医学賞受賞

### Thomas Seeley（2010）- 蜂の意思決定
- 著書 "Honeybee Democracy" で分蜂時の巣場所選択プロセスを詳細に解明
- 民主的な合意形成の生物学的モデルとして注目

### Bonabeau et al.（1999）- 蜂コロニー最適化（BCO）
- ミツバチの採餌行動を最適化アルゴリズムに応用する理論的基盤を確立

### Karaboga（2005）- ABC アルゴリズム（Artificial Bee Colony）
- ミツバチのフォレッジング行動をモデル化した最適化アルゴリズム「ABC」を提案
- 論文: "An Idea Based on Honey Bee Swarm for Numerical Optimization"

## まとめ：蜂の行動システムの特徴

| 特徴 | 内容 |
|------|------|
| **高精度な情報伝達** | ワグルダンスによる方向・距離・品質の符号化 |
| **品質ベースの意思決定** | 良い選択肢に自然に収束するフィードバック機構 |
| **ネガティブフィードバック** | 停止信号による不良選択肢の排除 |
| **動的な役割変化** | 日齢やコロニーの需要に応じた柔軟な分業 |
| **分散型環境制御** | 巣内温度の局所的行動による集合的調節 |
