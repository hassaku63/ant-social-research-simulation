# 調査タスク: Ant Colony Optimization (ACO)

## タスク概要

**目標**: ACOアルゴリズムの理論的基盤・主要バリアント・応用事例を体系的に整理し、テーマ2のシミュレーション実装に繋がる知識基盤を構築する。

**スコープ**:
- 調査すること:
  - ACOの生物学的背景（蟻のフェロモン経路探索メカニズム）
  - Dorigoによる原著研究とAnt System (AS) の定式化
  - 主要バリアント（ACS, MMAS, Elitist AS 等）の特徴と改良点
  - ACOの数学的・理論的性質（収束性、確率的最適化との関係）
  - 代表的な応用事例（TSP、車両配送問題、ネットワークルーティング等）
  - ACOの強み・限界
- 調査しないこと（除外事項）:
  - ACO以外の群知能アルゴリズム（PSO等）の詳細比較（概要レベルの言及は可）
  - 実装コードの作成（テーマ2で実施）
