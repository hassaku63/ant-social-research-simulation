# 調査タスク: スティグマジー（Stigmergy）

## タスク概要

**目標**: スティグマジーの概念・メカニズム・分類を体系的に整理し、ACOの理論的基盤の理解を深めるとともに、ソフトウェア・人間社会への応用事例を明らかにする。

**スコープ**:
- 調査すること:
  - スティグマジーの定義と歴史的背景（Grassé 1959）
  - スティグマジーの分類（定量的・定性的、マーカーベース・セマテクトニック）
  - 社会性昆虫におけるスティグマジーの具体例
  - ソフトウェア・インターネットにおけるスティグマジー応用
  - 人間社会におけるスティグマジーの事例
  - ACOとスティグマジーの関係
- 調査しないこと（除外事項）:
  - ACOのアルゴリズム詳細（ACO調査ドキュメントで完了済み）
  - 個別の応用システムの実装詳細
