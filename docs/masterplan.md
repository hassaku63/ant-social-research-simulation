# マスタープラン

このドキュメントはプロジェクト全体の目標・計画・進捗を概要レベルで管理するマスタープランです。
新しい作業を開始する際はこのドキュメントを参照し、完了時にステータスを更新してください。

---

## プロジェクト概要

社会性昆虫（蟻・蜂）の集団知性（Swarm Intelligence）を研究・応用するプロジェクト。
個体の単純なルールから集団の高効率な行動が生まれるメカニズムを解明し、
人間社会やソフトウェアへの応用を探ることが目的。
